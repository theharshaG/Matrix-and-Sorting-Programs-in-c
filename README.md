# Matrix-and-Sorting-Programs-in-c

## Overview

This repository contains C programs demonstrating matrix operations and sorting algorithms. These programs help in understanding two-dimensional arrays, matrix processing, nested loops, and sorting techniques commonly used in programming fundamentals.

The programs are useful for:

- Beginners learning C programming
- College laboratory exercises
- Coding interview preparation
- Logic-building practice
- Understanding array and matrix concepts

---

## Programs Included

### 1. Matrix Multiplication Program

This program multiplies two 2×2 matrices and displays the result matrix.

#### Output Example

```text
Enter first matrix:

1 2
3 4

Enter second matrix:

5 6
7 8

Result Matrix:

19 22
43 50
```

#### Concepts Used

- Two-dimensional arrays
- Matrix multiplication
- Nested loops
- Mathematical operations

---

### 2. Bubble Sort (Ascending Order)

This program sorts array elements in ascending order using the Bubble Sort algorithm.

#### Output Example

```text
Enter 5 elements:

50
20
40
10
30

Sorted Array:

10 20 30 40 50
```

#### Concepts Used

- Arrays
- Bubble Sort
- Nested loops
- Swapping technique

---

### 3. Bubble Sort (Descending Order with Largest and Second Largest)

This program sorts elements in descending order and displays the largest and second largest values.

#### Output Example

```text
Enter the array elements:

20
50
10
40
30

Descending order:

50 40 30 20 10

Largest = 50
Second Largest = 40
```

#### Concepts Used

- Arrays
- Bubble Sort
- Descending sorting
- Largest element logic
